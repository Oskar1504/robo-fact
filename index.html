                        moveX:1         | calls intern moveX function with parameter 1 => moves on tile to the right
                        
                        moveX:-1        | calls intern moveX function with parameter -1 => moves on tile to the left
                        
                        let:x:2         | defines/instanciates variable x with value 2
                        moveX:x         | calls intern moveX function with parameter 2 => moves two tiles to the right
                        
                        let:x:4         | defines/instanciates variable x with value 4
                        -:x:2           | removes 2 from variable x
                        moveX:x         | calls intern moveX function with parameter 2 => moves two tiles to the right
                        
                        let:x:0         | defines/instanciates variable x with value 0
                        +:x:2           | add 2 to variable x
                        moveX:x         | calls intern moveX function with parameter 2 => moves two tiles to the right
                        
                        let:x:0         | defines/instanciates variable x with value 0
                        set:x:2         | set variable x to 2
                        moveX:x         | calls intern moveX function with parameter 2 => moves two tiles to the right
                        
                        let:x:1         | defines/instanciates variable x with value 1
                        fnc:moveRight   | defines function with name | used with do:moveRight to jump to this line
                        moveX:1         | calls intern moveX function with parameter 1 => moves one tile to the right
                        +:x:1           | add 1 to variable x
                        do:moveRight    | jump to fnc:moveRight line

                        IMPORTANT THIS WILL OF COURSE END IN AN INFINITE LOOP. USE "force break execution" BUTTON TO STOP SCRIPT
